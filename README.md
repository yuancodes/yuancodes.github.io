## 本地预览

```bash
# 第一步：生成侧边栏（每次新增/删除文章后必须执行）
node gensidebar.js

# 方式一：docsify-cli（推荐）
docsify serve docs

# 方式二：Python 内置服务器
python -m http.server 3000 --directory docs

# 方式三：Node 全局安装
npx serve docs
```

> **注意**：`gensidebar.js` 会扫描 `docs/` 目录结构并自动生成 `docs/_sidebar.md`。每次增删文章后需先运行该命令，再启动预览服务。
>
> 启动后在浏览器打开 `http://localhost:3000` 即可浏览完整博客。
