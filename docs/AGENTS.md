# AGENTS.md

## 仓库概述

这是一个中文技术博客仓库，使用 Obsidian 管理。纯 Markdown 内容，无代码、无构建系统、无测试。

## 目录结构

顶级目录格式为 `NN_主题/`（数字前缀 + 中文类别名）。子目录使用 `NN_子主题/` 组织系列文章。

主要分类：
- `00_先利其器` — 工具（IDE、数据库工具）
- `01_操作系统` — 系统（Linux、Windows、网络）
- `04_大前端` — 前端（Vue、React、Bootstrap、JSP）
- `05_数据库` — 数据库（MySQL、Redis、MongoDB）
- `08_框架技术` — 框架（Spring Boot、MyBatis）
- `10_设计模式` — 设计模式
- `12_项目管理` — 项目管理（Maven、Docker、PMP）
- `15_分布式` — 分布式系统
- `16_性能优化` — 性能优化
- `17_项目设计` — 项目设计
- `21_代码片段` — 代码片段（Java 工具类）
- `22_AI` — AI 工具与提示词
- `99_个人成长` — 个人成长

## 文件命名规范

格式为 `NN-标题.md`（数字前缀 + 中文标题 + `.md`）。

示例：
- `01-Redis 环境搭建.md`
- `02-Redis 数据存储类型.md`

同一系列文章按顺序编号。

## 编辑规则

- 所有内容使用简体中文
- 保持现有格式风格，不要转换为英文或重构结构
- 新文章遵循 `NN-标题.md` 命名模式
- 将文章放置在对应的分类目录中
- Obsidian 是主要编辑器，`.obsidian/` 配置存在，需尊重
